# Gatehouse - Parking Lot Management System

A full-stack parking management application.

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Features

- Vehicle parking
- Ticket generation
- Slot availability tracking
- Vehicle exit
- Automatic fare calculation
- Parked vehicle list
